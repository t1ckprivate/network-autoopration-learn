## 如何使用？
1. 下载本仓库，并使用IDE打开./2-Telnetlib文件夹
2. 打开topo中的pro3.topo，**无需做任何设置，本仓库下的所有拓扑均已配置完成**
3. 在IDE中使得终端位于./2-Telnetlib中，运行code.py

## ⚠️ 注意 ！
- 不建议抓包，否则vmnet会炸，需恢复vmnet设置+重启

- 宿主机与ENSP之间没有路由，导致无法互通，建议添加：
```
route -p add 10.0.0.0 mask 255.0.0.0 10.3.1.254
```

- 虚拟机与ENSP之间没有路由，导致无法互通，建议添加：
```
sudo ip route add 10.0.0.0/8 via 10.3.1.254
```

## ❕ 提示
- 老师提供的虚拟机python版本为3.6，不支持telnetlib，建议安装3.12.10
